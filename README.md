# literate-umbrella
En privat, lösenordsfri webbapp där du och dina vänner delar och återanvänder AI-resurser — prompts, system prompts, SKILL.md-filer och workflows/agentkedjor. Åtkomst sker via delad länk (ingen inloggning). All data lagras som JSON-filer i repot, och appen hostas statiskt via GitHub Pages
